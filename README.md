# Sensor Build

Уебсайт за Sensor Build - майсторски услуги.

## Стартиране на проекта

За да стартирате development сървъра:

```bash
npm run dev
```

Отворете [http://localhost:3000](http://localhost:3000) в браузъра.

## Build

За да build-нете проекта за production:

```bash
npm run build
npm start
```
